Código de Preprocesamiento

# Preprocesamiento de imágenes
IMG_WIDTH = 64
IMG_HEIGHT = 64

def preprocess_image(image, label):
  # Redimensionar imagen
  image = tf.image.resize(image, (IMG_WIDTH, IMG_HEIGHT))
  # Aplanar imagen
  image = tf.reshape(image, [-1])
  # Normalizar imagen
  image = image / 255.0
  return image, label

Conclusiones y Aprendizajes

El proyecto demostró que es posible implementar un clasificador básico de imágenes usando algoritmos tradicionales de Machine Learning, sentando las bases para explorar técnicas más avanzadas como Redes Neuronales Convolucionales.

Logros Principales:

  • Implementación exitosa del pipeline completo de ML
  • Creación de interfaz para predicción de nuevas imágenes
  • Análisis detallado del rendimiento del modelo
  • Identificación de áreas de mejora para futuras iteraciones

Próximos Pasos:

  • Implementar Redes Neuronales Convolucionales (CNN)
  • Experimentar con Transfer Learning
  • Aplicar técnicas de aumento de datos
  • Optimizar hiperparámetros del modelo
